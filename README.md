# 🧠 Brain Tumor Detection using Deep Learning

## 📌 Overview

This project focuses on detecting brain tumors from MRI images using Deep Learning techniques. The model automatically classifies MRI scans into **Tumor** or **No Tumor**, assisting in faster and more accurate diagnosis.

---

## 🎯 Problem Statement

Manual analysis of brain MRI scans is time-consuming and requires expert radiologists. The objective of this project is to build an automated deep learning system that can classify MRI images and detect the presence of brain tumors.

---

## 🗂️ Dataset

* **Source:** Kaggle
* **Link:** https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection
* **Classes:**

  * Yes (Tumor)
  * No (No Tumor)
